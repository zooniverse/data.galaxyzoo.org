Galaxy Zoo: Mergers - Data

This page contains links to files with data from the Galaxy Zoo: Mergers project. There are files describing the interacting galaxy systems as well as sets of simulation input parameters.



Target Groups and Interacting Pairs

To see more information about each of the morphological groups of targets and pairs of galaxies, visit the Targets page.



Summary Data

The following files contain summary information about the entire set of targets. The Target Info file contains the RA/Dec for the center of each galaxy in the pair as well as a set of alternate names drawn from Arp, NGC/IC, UGC, SDSS, and 2MASS. The Targets SDSS file contains the id as well as the dered magnitudes for u, g, r, i, and z channels for both the primary and secondary of each target. Where available, both the DR7 and DR8 values are included.

Tables from the Data Release Paper

There are two tables in the data release paper that provide details about each target. Table 1 provides location and alias information. Table 4 provides summary information about the best-fit simulation parameters. They can be downloaded in tab-delimited files below.
